🔧 Troubleshooting
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Problema: Porta 8000 já em uso
✅ Solução: 
   - Feche o outro servidor
   - Ou edite start-server.bat e mude "8000" para "8080"

❌ Problema: "python is not recognized"
✅ Solução:
   - Instale Python de python.org
   - Assinale "Add Python to PATH"

❌ Problema: CSS não carrega
✅ Solução:
   - Pressione Ctrl+Shift+Delete (limpar cache)
   - Recarregue a página com F5

❌ Problema: Login não funciona
✅ Solução:
   - Login é simulado - apenas selecione o tipo
   - Dados guardados em localStorage

📖 Editar o Projeto
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Edite os ficheiros com um editor (VS Code, Sublime, etc)
2. Guarde as alterações (Ctrl+S)
3. Recarregue o navegador (F5)
4. Veja as mudanças em tempo real!

Para adicionar eventos:
  Edite: data.js → mockEvents

Para mudar cores:
  Edite: styles.css → vars no :root

Para mudar textos:
  Edite: os ficheiros .html
